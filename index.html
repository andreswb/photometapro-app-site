Frequently Asked Questions

How do I add photos?

Use ⌘O to open a file picker, or drag and drop photos directly into the app. Supported formats include JPEG, PNG, TIFF, HEIC, and many RAW formats (compatibility varies by macOS version).

Are my originals ever modified?

Never. PhotoMetaPro always writes processed files to a separate output directory. Your originals remain completely untouched.

How does AI keyword suggestion work?

PhotoMetaPro uses Apple's Vision framework to analyse your photos directly on your Mac — no internet connection, no subscription and no cost per use. It suggests up to 15 relevant keywords based on the image content, filtered to exclude words already in your template.

What is the Stock Readiness Score?

It checks your template's metadata (title, keywords, copyright, description and more) against the submission requirements of major stock agencies. Each field is shown as pass, warn or fail so you can fix gaps before submitting your photos.

How does Two-Camera Time Sync work?

Select one reference photo from each camera. PhotoMetaPro calculates the exact timestamp offset between the two cameras and applies the correction to all photos from the secondary camera, so both streams are perfectly aligned.

What does Delivery Package do?

It exports a client-ready folder from your processed photos. You can optionally add a text watermark, strip GPS data for privacy, choose a flat or date/client folder structure, and generate a file manifest. Useful for handing off work to clients or agencies.

How do I use the Keyword Library?

Open the Keyword Library from the toolbar. Add individual keywords or entire sets, then click to apply them to your current template. It's designed for keywords you reuse across multiple projects — genres, locations, client names.

How does the Watch Folder work?

Enable it in the toolbar and select a folder to monitor. Whenever new photos are added to that folder, PhotoMetaPro automatically picks them up and processes them with your current template settings.

How do I import GPS from a GPX file?

Use the GPX Import button in the toolbar. Select your .gpx track file and PhotoMetaPro matches each photo's timestamp to the nearest track point, automatically tagging GPS coordinates across the whole shoot.

What is Date & Time Shift?

Use it to correct timestamp drift across a shoot — for example when you forgot to set the correct time zone, or when your camera clock was off. Enter the offset (hours, minutes, seconds) and it is applied to all selected photos.

How does Batch Rename work?

Open the Batch Rename sheet from the toolbar. Choose a pattern (prefix, sequence number, date, or custom), preview the result live, and apply to all or selected photos.

What are XMP sidecar files?

Some RAW formats (e.g. certain Canon and Nikon RAW files) do not support metadata embedded directly in the file. For these, PhotoMetaPro can write a companion .xmp sidecar file that stores the metadata alongside the original without modifying it.

How do I undo a change?

Press ⌘Z to undo or ⌘⇧Z to redo. PhotoMetaPro keeps a 20-level history for all template and metadata changes.

What happens if the app quits unexpectedly?

PhotoMetaPro auto-saves your session state every 10 photos. On the next launch it detects the interrupted session and offers to resume from where it left off.

Can I pause processing?

Yes. Click the Cancel button during processing and you'll get three options: skip the current photo and continue, pause (resume later), or stop the entire batch.

Does the app need an internet connection?

Only for the optional reverse geocoding feature, which converts GPS coordinates into readable place names using Apple's location services. Every other feature works completely offline.

What formats are supported?

JPEG, PNG, TIFF and HEIC are fully supported. Many RAW formats are also supported (CR2, CR3, NEF, ARW, DNG and others), though metadata support for some RAW formats may be limited depending on camera model and macOS version.

Is there a free trial?

PhotoMetaPro is a paid app with no separate trial version. The Mac App Store does not currently support trials for Mac apps. All features, screenshots and this FAQ are available to review before purchasing. If you have specific pre-purchase questions, send us an email and we'll be happy to help.

What happens to existing metadata already in my photos?

Existing metadata is preserved by default. PhotoMetaPro adds or overwrites only the fields you explicitly configure in your template. Fields left empty in the template are not touched. Original EXIF data captured by your camera — shutter speed, aperture, ISO, lens information — is always preserved.

How do I set an output directory?

Click the folder icon next to the output path field in the main window and choose your destination folder. PhotoMetaPro remembers the selected directory per template using secure bookmarks, so you only need to set it once per template.

How do I export or share a template?

Click the Templates menu in the toolbar and choose Export Template. This saves a .json file you can back up or share with colleagues. Use Import Template from the same menu to load it on another Mac. Templates carry all metadata fields, output settings and filename preferences.

What keyboard shortcuts are available?

⌘O opens the file picker, ⌘P starts processing, ⌘A selects all photos, ⌘Z undoes the last change, ⌘⇧Z redoes it, and Escape dismisses most sheets. A complete list is available inside the app under Help → Keyboard Shortcuts.

Does PhotoMetaPro support Apple Silicon?

Yes. PhotoMetaPro is a universal binary — it runs natively on both Apple Silicon (M1, M2, M3, M4 and later) and Intel Macs. No Rosetta translation is required on Apple Silicon machines.

Why did some of my RAW files not receive metadata?

Some proprietary RAW formats do not support embedded metadata. For these files, enable XMP Sidecar output in the template settings. PhotoMetaPro will create a companion .xmp file alongside each original, which Lightroom, Capture One, Photoshop and most professional applications read automatically.

How do I view the processing log?

Click the View Log button in the main window after a processing run. The log is filterable by status (success, warning, error, info) and can be exported as a plain text file — useful for auditing large batches or providing processing summaries to clients.

Häufige Fragen

Wie füge ich Fotos hinzu?

Mit ⌘O eine Dateiauswahl öffnen oder Fotos per Drag-and-Drop direkt in die App ziehen. Unterstützte Formate: JPEG, PNG, TIFF, HEIC und viele RAW-Formate (Kompatibilität je nach macOS-Version).

Werden meine Originaldateien verändert?

Nein, niemals. PhotoMetaPro schreibt verarbeitete Dateien immer in ein separates Ausgabeverzeichnis. Die Originale bleiben vollständig unberührt.

Wie funktionieren die KI-Keyword-Vorschläge?

PhotoMetaPro nutzt Apples Vision-Framework zur Bildanalyse direkt auf deinem Mac — ohne Internetverbindung, ohne Abo und ohne Kosten pro Nutzung. Es schlägt bis zu 15 relevante Keywords basierend auf dem Bildinhalt vor, gefiltert um bereits in der Vorlage vorhandene Begriffe.

Was ist der Stock-Readiness-Score?

Er prüft die Metadaten deiner Vorlage (Titel, Keywords, Copyright, Beschreibung u. a.) gegen die Einreichungsanforderungen großer Stockagenturen. Jedes Feld wird als bestanden, Warnung oder fehlgeschlagen angezeigt, damit du Lücken vor dem Einreichen beheben kannst.

Wie funktioniert der Zwei-Kamera-Zeitsync?

Je ein Referenzfoto pro Kamera auswählen. PhotoMetaPro berechnet den genauen Zeitversatz zwischen den beiden Kameras und wendet die Korrektur auf alle Fotos der zweiten Kamera an, sodass beide Streams perfekt synchronisiert sind.

Was macht die Lieferpaket-Funktion?

Sie exportiert einen kundenfertig aufbereiteten Ordner aus deinen verarbeiteten Fotos. Optional mit Wasserzeichen, GPS-Entfernung für den Datenschutz, flacher oder Datum/Kunden-Ordnerstruktur und automatischem Dateimanifest. Ideal für die Übergabe an Kunden oder Agenturen.

Wie nutze ich die Keyword-Bibliothek?

Die Keyword-Bibliothek über die Symbolleiste öffnen. Einzelne Keywords oder ganze Sets hinzufügen und per Klick auf die aktuelle Vorlage anwenden. Gedacht für Keywords, die über mehrere Projekte hinweg verwendet werden — Genres, Orte, Kundennamen.

Wie funktioniert der Watch-Ordner?

In der Symbolleiste aktivieren und einen zu überwachenden Ordner auswählen. Sobald neue Fotos in diesem Ordner auftauchen, erkennt PhotoMetaPro sie automatisch und verarbeitet sie mit den aktuellen Vorlagen-Einstellungen.

Wie importiere ich GPS aus einer GPX-Datei?

Die GPX-Import-Schaltfläche in der Symbolleiste verwenden. Nach Auswahl der .gpx-Trackdatei ordnet PhotoMetaPro jedem Foto den nächstgelegenen Trackpunkt anhand des Zeitstempels zu und taggt GPS-Koordinaten automatisch über den gesamten Shoot.

Was ist die Datum-&-Zeit-Verschiebung?

Damit lassen sich Zeitstempel-Abweichungen eines ganzen Shoots korrigieren — z. B. wenn die Zeitzone vergessen oder die Kamerazeit falsch eingestellt war. Offset (Stunden, Minuten, Sekunden) eingeben und er wird auf alle ausgewählten Fotos angewendet.

Wie funktioniert die Stapelumbenennung?

Das Stapelumbenennen-Sheet über die Symbolleiste öffnen. Muster wählen (Präfix, Sequenznummer, Datum oder benutzerdefiniert), das Ergebnis live in der Vorschau sehen und auf alle oder ausgewählte Fotos anwenden.

Was sind XMP-Sidecar-Dateien?

Einige RAW-Formate unterstützen keine direkt eingebetteten Metadaten. Für diese kann PhotoMetaPro eine .xmp-Begleitdatei erstellen, die die Metadaten neben dem Original speichert, ohne es zu verändern.

Wie kann ich eine Änderung rückgängig machen?

⌘Z zum Rückgängigmachen, ⌘⇧Z zum Wiederholen. PhotoMetaPro speichert einen 20-stufigen Verlauf für alle Vorlagen- und Metadaten-Änderungen.

PhotoMetaPro sichert den Sitzungsstatus automatisch alle 10 Fotos. Beim nächsten Start erkennt die App die unterbrochene Sitzung und bietet an, dort weiterzumachen, wo aufgehört wurde.

Ja. Die Abbrechen-Schaltfläche während der Verarbeitung anklicken — es erscheinen drei Optionen: aktuelles Foto überspringen und fortfahren, pausieren (später fortsetzen) oder den gesamten Stapel stoppen.

Privacy Policy

No Data Collection

PhotoMetaPro does not collect, transmit, store remotely, or share any data about you or your photos. All processing happens entirely on your Mac.

What Is Stored Locally

The following data is stored only on your device using macOS standard mechanisms (UserDefaults, app sandbox, file system) and never leaves your Mac:

  • Project templates (copyright text, keywords, client info, output settings)
  • Keyword Library entries
  • Output directory bookmarks (sandboxed security-scoped)
  • Processing logs and statistics
  • Crash and session recovery state (automatically deleted after 24 hours)
  • App preferences (language, last-used settings)
  • Watch Folder configuration

Network Usage

PhotoMetaPro makes no network requests except for the optional Reverse Geocoding feature. When you add GPS coordinates and enable reverse geocoding, your Mac contacts Apple's location services to resolve coordinates into place names. This uses the same Apple CoreLocation infrastructure as the macOS Maps app. No other feature makes any network request.

The AI Keyword Suggestions feature uses Apple's Vision framework and runs entirely on-device. No image data or keywords are sent anywhere.

No Analytics or Tracking

PhotoMetaPro contains zero third-party SDKs, analytics libraries, crash reporters, or telemetry. No usage statistics, device identifiers, or diagnostics are collected or transmitted.

No Advertising

PhotoMetaPro contains no advertising and does not use any advertising identifier or tracking framework.

Your Photo Data

PhotoMetaPro reads photo files only to display thumbnails and write metadata. No image content is uploaded, cached remotely, or shared with any service. Processed files are written to the output directory you choose.

PhotoMetaPro liest Fotodateien ausschließlich zur Anzeige von Vorschaubildern und zum Schreiben von Metadaten. Kein Bildinhalt wird hochgeladen, remote zwischengespeichert oder an einen Dienst weitergegeben. Verarbeitete Dateien werden in das von dir gewählte Ausgabeverzeichnis geschrieben.

Diese Richtlinie kann aktualisiert werden, um neue Funktionen widerzuspiegeln. Das unten stehende Datum zeigt die letzte Überarbeitung. Die weitere Nutzung der App nach einer Aktualisierung gilt als Zustimmung zur geänderten Richtlinie.
